## Failure found

The push-triggered GitHub Actions run for the S041 commit failed in `tests/unit/google-identity.test.ts`. The forged-token test changed only the trailing characters of a base64url signature. That mutation was not guaranteed to change the decoded RSA signature bytes because trailing base64url bits can be non-significant, so the test could intermittently accept a still-valid token. The verifier itself was not changed.

## Correction

The test now signs the otherwise-valid token with a separately generated RSA private key while the token continues to identify the trusted test JWKS key. This makes the forged signature deterministic and preserves the intended assertion that verification fails closed.
