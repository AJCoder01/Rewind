# S013 CI and repository security evidence — 2026-07-15

## Scope

S013 adds a locked Node 24 GitHub Actions job covering install, lint, strict typecheck, unit/contract tests, production build, dependency audit, tracked-file secret scanning, production fake-mode rejection, ephemeral PostgreSQL migration apply/replay, and the fixture browser smoke.

The scan reads only Git-tracked files. It reports file/rule identifiers and never prints matched content. CI's PostgreSQL service is an isolated ephemeral test service with trust authentication; it is not a provider or demo environment.

## Environment limitation

The local machine has no Docker or PostgreSQL executable, so the ephemeral migration job was not run locally. `npm run db:verify:ephemeral` was invoked with its URL unset and failed closed before any connection. The committed CI workflow runs the apply/replay check against a disposable PostgreSQL 16 service.

## Remaining risk

The hosted CI job must complete after the branch is pushed. S014–S017 remain open, and no provider credentials or live integrations were accessed.
